Cat videos (from the Cat Gatekeeper extension package)
=======================================================

Place these two files next to this README:

  neko1.webm  — intro clip (plays once)
  neko2.webm  — looping “sleep” clip

Copy them from a **full** unpack of the extension (your Chrome profile folder
often has them under):

  Chrome:  …/Chrome/Default/Extensions/elbikiflgfhjdjmficnigpeegjbhdidh/<version>/assets/

Or re-download / unpack the `.crx` so the `assets/` folder is included.

The break screen expects both files so the cat videos can play.
